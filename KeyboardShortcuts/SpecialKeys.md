## Frequently used symbols

- <kbd>⌘</kbd> Command
- <kbd>⌃</kbd> Control
- <kbd>⌥</kbd> Option
- <kbd>⇧</kbd> Shift
- <kbd>⇪</kbd> Caps lock
- <kbd>⏎</kbd> Return
- <kbd>⎋</kbd> Escape
- <kbd>⇥</kbd> Tab
- <kbd>↑</kbd> Up
- <kbd>↓</kbd> Down
- <kbd>←</kbd> Left
- <kbd>→</kbd> Right
